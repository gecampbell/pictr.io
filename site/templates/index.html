Pictr anonymous, ephemeral picture sharing

Pictr lets you share pictures with the world, but only for a limited time. After a selected interval, they will simply disappear. If someone loves the picture, however, it might stay around longer. Likewise, if someone hates the picture, its time remaining will be shortened. If you still have questions, check the FAQ.

Pictr is in alpha, and it might be unstable from time to time. You can report problems by creating an issue in our Github repository.
